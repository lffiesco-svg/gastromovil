# GastroWeb: Plataforma de Domicilios Gastronómicos

GastroWeb es una aplicación web de domicilios gastronómicos desarrollada con **Django**, diseñada para conectar clientes, restaurantes y domiciliarios en una sola plataforma digital, intuitiva y segura.

## ¿Qué es GastroWeb?

GastroWeb nació como respuesta a la necesidad de modernizar el sector gastronómico de **Garzón, Huila, Colombia**, donde muchos restaurantes carecen de herramientas digitales para gestionar sus domicilios de manera eficiente.

La plataforma permite a los usuarios realizar pedidos de alimentos a restaurantes registrados, con seguimiento en tiempo real, pagos en línea y un asistente de inteligencia artificial que sugiere restaurantes y platos según las preferencias del usuario en lenguaje natural.

## Tecnologías utilizadas

| Tecnología | Versión | Uso |
|---|---|---|
| Django | 6.0.3 | Framework principal del backend |
| Django REST Framework | 3.17 | APIs RESTful |
| Simple JWT | 5.5.1 | Autenticación segura |
| Groq + LangChain | Latest | Asistente de IA |
| Django Channels | 4.3.2 | Tiempo real / WebSockets |
| Daphne | 4.2.1 | Servidor ASGI |
| Cloudinary | Latest | Almacenamiento de imágenes |
| Resend | 2.30.1 | Correos transaccionales |
| MySQL | Latest | Base de datos |
| Tailwind CSS | Latest | Diseño responsivo |
| Railway | Latest | Despliegue en producción |
